Releases

The current release is 1.7.1 and was published on July 30, 2017. The release note for this release can be found here.

The following distributions are available for download:

  Link Checksums and signatures
Binary distribution rampart-dist-1.7.1-bin.zip MD5 SHA1 PGP
Source distribution rampart-dist-1.7.1-src.zip MD5 SHA1 PGP

The binary distribution contains the following Axis2 modules:

  • rampart-1.7.1.mar provides support for WS-Security and WS-SecureConversation features.

  • rahas-1.7.1.mar provides the necessary components to enable SecurityTokenService functionality on a service.

The signatures of the distributions can be verified against the public keys in the KEYS file.

Maintenance releases from branches other than the main branch can be found here. Distributions for older releases can be found in the archive.

All releases are also available as Maven artifacts in the central repository.
