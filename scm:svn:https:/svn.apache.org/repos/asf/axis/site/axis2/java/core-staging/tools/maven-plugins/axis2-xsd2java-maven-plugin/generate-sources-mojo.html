axis2-xsd2java:generate-sources

Full name:

org.apache.axis2:axis2-xsd2java-maven-plugin:1.8.2:generate-sources

Description:

Generates Java classes from the specified schema files.

Attributes:

  • Requires a Maven project to be executed.
  • The goal is thread-safe and supports parallel builds.
  • Binds by default to the lifecycle phase: generate-sources.

Required Parameters

Name Type Since Description
<xsdFiles> File[] - The list of XSD files for which to generate the Java code.

Optional Parameters

Name Type Since Description
<helperMode> boolean - (no description)
<ignoreUnexpected> boolean - Specifies whether unexpected elements should be ignored (log warning) instead of creating an exception.
<mapperClassPackage> String - (no description)
<namespaceMappings> NamespaceMapping[] - Mapping of namespaces to target Java packages.
<noNamespacePackageName> String - The Java package to use for schema items without namespace.
<outputDirectory> File - The output directory for the generated Java code.
Default value is: ${project.build.directory}/generated-sources/xsd2java.
<packageName> String - (no description)
